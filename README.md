# coin-info 🪙 💰 📊
A simple web app that displays data on a user inputted crypto currency using the coincap api. </br>

Data includes name, symbol, current price, and 24 hour volume. </br>

Live site ⬇️: </br>

https://kmal808.github.io/coin-info/ </br>
